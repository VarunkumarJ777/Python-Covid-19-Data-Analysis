# COVID-19 Impact Data Analysis
Leveraging daily trade data and the power of Python, Pandas, and Matplotlib, this analysis explores the impact of COVID-19 on trade trends until December 2021. It examines annual patterns, commodity influences, transport mode effects, and specific country responses.

## Dataset

This "Effects of COVID-19 on trade" dataset provides provisional data on daily trade activity as of December 15, 2021. It includes insights into trade direction, dates, countries involved, commodity types, transport modes, and trade values.

## Analyses Performed

1. **Overall Trade Trend**: Visualized the increasing trade value over the years 2019, 2020, and 2021.
2. **Exports vs. Imports**: Compared the cumulative exports and imports over the mentioned years.
3. **Top Commodities Affected**: Identified commodities with the most significant changes in trade value.
4. **Transport Mode Analysis**: Analyzed the effect on different modes of transport.
5. **Country Analysis**: Checked the trade impact on specific countries.

## Tools Used:

- **Python**: The primary language for data processing and analysis.
- **Pandas**: Used for data manipulation and analysis.
- **Matplotlib**: For data visualization and creating charts.
